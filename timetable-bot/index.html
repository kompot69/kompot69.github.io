Бот для ВК для получения расписания с сайта ПТПИТ

Что умеет? / Какие команды есть?

Команды получения расписания:

Регистр значения не имеет

Расписание/пары сегодня/завтра

Покажет расписание на выбранный день, если это суббота - покажет на понедельник.
Пример: Пары на завтра
Пример: Расписание сегодня
Не используйте сочетание "Распиание на"!

Пара/пары сейчас/следующие(ая)

Пары: покажет оставшиеся пары. ; Пара: покажет текущую пару.
Пример: Какая сейчас пара
Пример: Следующие пары

Расписание на д.м.гг

Покажет расписание на выбранную дату
Пример: Расписание на 07.11.21

Остальные команды:

👮 - команда для админов
👥 - команда для всех пользователей

Бот группа 👥

Устанавливает группу для чата.
Пример: Бот группа 18сзи1

Бот помощь 👥

Покажет список всех доступных пользователю команд.

Добавить админа 👮

Добавляет ID пользователя в список админов.
Пример: Добавить админа 472153219

Бот статус 👥 👮

Покажет коды состояния сайтов, ID чата и установленную группу.
Покажет кол-во админов и время системы. 👮

Бот рестарт 👮

Выход из бота для последующего его запуска контроллером.

Установка (Linux)

1. Скачать python-бота

2. Установить необходимые python-пакеты из requirements.txt

3. Запустить botController.py и следовать инструкциям в консоли

Деплой на хостинг (Deploy-F)

Доступно с версии 0.3

1. Выполнить устновку на локальной машине

2. Заархивировать все созданые файлы бота

3. Залить готовый архив на хостинг
